MDSlides README
===================

A Web-based Markdown to slideshow program

#Try it online
MDSlides is live on heroku: [MDSlides Online](http://mdslides.herokuapp.com/)

#Acknowledgements

MDSlides is based on [reveal.js](https://github.com/hakimel/reveal.js/)
